Employee Management System (EMS)
Java + Text File Version

This folder contains the code for the Employee Management System.

Files included:
- Main.java
- employees.txt
- attendance.txt
- README1.txt

How to run:
1. Open the project in NetBeans or another Java IDE.
2. Compile Main.java
3. Run Main.java
4. Use the menu to:
   - Add Employee
   - View Employees
   - Record Attendance
   - View Attendance

How the system saves:
This version uses Java text files.
- employees.txt saves employee names
- attendance.txt saves attendance records

This means the data can still be there after the program closes.
